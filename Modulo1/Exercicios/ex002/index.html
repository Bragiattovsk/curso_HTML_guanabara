Parágrafos e quebras de linhas

Você pode escrever um texto com várias linhas e quebras de linhas. Para isso, basta usar a tag <p> para criar parágrafos e a tag <br> para criar quebras de linhas.

Se precisar de uma quebra de linha deve-se utilizar a tag <br> para que o texto seja exibido em uma nova linha.
